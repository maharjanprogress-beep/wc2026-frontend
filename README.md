# WC 2026 AI Predictor — Mobile App

Expo React Native app for the WC 2026 AI Predictor.

## Setup

```bash
npm install
```

Set your backend URL in `.env`:
```
EXPO_PUBLIC_DOMAIN=your-render-backend.onrender.com
```

## Run

```bash
npx expo start
```

Scan the QR code with Expo Go on your phone.

## Build for App Store / Google Play

```bash
npx eas build --platform all
```
